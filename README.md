# ThisReply

AI-powered dating conversation helper. Send the right reply every time.

## Getting Started

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## Tech Stack

- Next.js 14
- TypeScript
- Tailwind CSS
- Supabase (Auth & Database)
- Stripe (Payments)
- OpenAI (Vision API)
